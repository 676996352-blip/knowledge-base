## 2. 底层认知

### 核心观点

> **没有标准答案**: 航海和深海圈都不存在"做了就能赚钱"的标准SOP

### 原因分析

| 现状 | 后果 |
|------|------|
| AI模型能力已非常强大 | YouTube保护内容生态 |
| 很多人习惯"喂饭式"学习 | 不再允许照抄教程就能赚钱 |
| 红利已被大量复制 | 校色史违规严重 |

### 正确心态

```
赚钱 = 基本功 + 心力 + 愿力
     ≠ 等着人喂饭
```

### 关键认知

1. **扎实基本功** - 无论什么机会，都需要扎实的内容创作和内容理解能力
2. **主动学习** - 不能只等教程，要自己探索和实践
3. **理解平台** - YouTube不允许大量同质化内容存在

## 3. 爆款方法论

### 最简公式

```
爆款视频 = 爆款IP + 爆款脚本
```

### 结构解析

| 要素 | 作用 | 说明 |
|------|------|------|
| **爆款IP** | 好皮囊 | 吸引眼球的形象 |
| **爆款脚本** | 好故事 | 触动情绪的内容 |

### 补充说明

- **爆款形式** 不等于 爆款IP
- 爆款形式还包括：可爱猫狗、治愈画风等
- 为了简化理解，这里只讲爆款IP

## 4. 爆款IP详解

### 4.1 什么是爆款IP

> **定义**: YouTube上百看不腻的好皮囊

### 常见类型

| 类型 | 示例 |
|------|------|
| 卡通动物 | 猫、狗 |
| 影视IP | 蜘蛛侠、钢铁侠 |
| AI形象 | AI3眼金、AI山海经 |
| 其他 | K-pop相关形象 |

### 4.2 IP流量规律

| 规律 | 说明 | 案例 |
|------|------|------|
| 有些IP持续有流量 | 长期稳定 | 蜘蛛侠(2年前17亿播放，现在还有大几千万) |
| 有些IP阶段性爆发 | 火一段时间后下降 | AI3眼金(去年4-7月很火，现在下降) |
| 国内火≠国外火 | 需要验证 | 我的刀盾(抖音火，油管没什么流量) |

### 4.3 寻找IP的三大渠道

#### 渠道一：梗百科

**地址**: 自行搜索"梗百科"

**使用方法**:
1. 每周查看更新内容
2. 关注最近爆火的内容
3. 去油管验证流量

**案例**: 我的刀盾 → 抖音很火 → 油管搜索只有300万播放(没流量)

#### 渠道二：无痕刷YouTube首页

**操作步骤**:
1. 打开无痕模式
2. 访问YouTube首页
3. 上下翻刷视频
4. 看到陌生形象就记录
5. 用谷歌以图搜图识别
6. 去油管搜索验证

**核心原理**:
```
上升期IP → 一定会被YouTube推荐 → 首页刷到
```

**注意事项**:
- 刷到视频不要点进去看(避免信息茧房)
- 复制链接到其他浏览器查看

#### 渠道三：关注全球热门影视游戏

**关注对象**:
- 热门电影/电视剧
- 热门游戏
- 全球流行文化

**案例**:

| IP | 来源 | 油管播放量 |
|-----|------|----------|
| 鱿鱼游戏 | 影视 | 亿级播放 |
| 我的世界 | 游戏 | 大几亿播放 |
| 鬼灭之刃 | 动漫 | 持续有量 |
| K-pop | 音乐 | 持续有量 |

### 4.4 判断IP能否使用

#### 验证步骤

1. **去油管搜索IP名称**
2. **按热门程度排序**
3. **筛选本月/本周数据**
4. **选择短视频**

#### 判断标准

| 情况 | 结论 | 行动 |
|------|------|------|
| 近期有百万+播放 | IP有流量 | 可以做 |
| 只有几万播放 | IP流量下降 | 谨慎做 |
| 基本没播放 | IP没流量 | 不建议做 |

#### 进阶搜索技巧

**方法1**: 用主角名称搜索
- 有些视频不用IP名字做标题
- 用主角名搜索可能找到

**方法2**: 通过标签搜索
- 点开视频 → 看蓝色标签
- 点击标签 → 查看相关视频播放量

### 4.5 围绕IP做内容的三层境界

| 层级 | 做法 | 难度 | 说明 |
|------|------|------|------|
| **第一层** | 借用IP外壳演绎脚本 | ⭐ | 内容与IP世界观无关 |
| **第二层** | IP与脚本轻度结合 | ⭐⭐ | 粉丝会感兴趣 |
| **第三层** | 深度参考IP背景制作 | ⭐⭐⭐⭐ | 高契合度内容 |

#### 举例

| 层级 | 案例 |
|------|------|
| 第一层 | 用蜘蛛侠形象演一个普通友情故事 |
| 第二层 | 蜘蛛侠的日常生活 |
| 第三层 | 蜘蛛侠的影视解说、游戏攻略 |

## 5. 爆款脚本详解

### 5.1 什么是爆款脚本

> **定义**: 能触动你情绪的好故事

### 好脚本 vs 坏脚本

| 好脚本 | 坏脚本 |
|--------|--------|
| 触动情绪 | 视觉洗脑 |
| 有深度内容 | 看完了无波动 |
| 爆了还会再爆 | 容易同质化 |

#### 案例对比

**坏脚本**: 鬼畜洗脑视频(视觉吸引，但无情绪波动)

**好脚本**: 感人故事(触动情绪，容易传播)

### 5.2 好脚本的验证方法

**特征**: 一个脚本用不同IP、不同形象做，都能爆

**案例**: iPhone手机脚本
- 2025年5月: 3亿播放
- 2025年6月(复刻): 3900万播放
- 2026年2月(复刻): 4000万播放

**结论**: 半年后还能爆，证明是好脚本

### 5.3 寻找脚本的三大方法

#### 方法一：让YouTube推荐真人博主

**操作步骤**:
1. 订阅提供的真人博主列表
2. 刷他们的视频
3. 回首页，让算法学习
4. 首页开始推同类博主
5. 持续关注新博主

**小技巧**: 通过短视频链接找长视频形式
```
短视频链接: youtube.com/shorts/XXX
长视频链接: youtube.com/watch?v=XXX
替换ID → 查看关联视频 → 发现更多博主
```

#### 方法二：第三方数据网站

**推荐网站**: (见文档链接)

**使用方法**:
1. 用谷歌账号登录
2. 选择Top Videos
3. 筛选日期(7天/28天/3个月)
4. 筛选视频类型(长视频/短视频)
5. 逐个点击查看内容

**用途**:
- 找爆款脚本
- 找赛道方向
- 发现新形式

#### 方法三：国内平台挖掘

**平台**: 抖音、快手、小红书、B站

**关键词**:
| 类型 | 关键词 |
|------|--------|
| 搞笑 | 搞笑反转、神转折 |
| 感人 | 很感人、结尾绝了 |
| 其他 | 暖心、催泪 |

**操作步骤**:
1. 用小号刷(避免信息茧房)
2. 关注同类型博主
3. 刷到感兴趣的脚本就关注
4. 点赞/不感兴趣训练算法
5. 同类脚本关注5-6个账号后，养号完成

**进阶**: 搜索"抽象"等标签发现新类型

## 6. 脚本筛选四原则

### 原则一：避开未成年人相关

**原因**: 油管审查特别严，抓到直接封号，无申诉机会

**禁止类型**:
- 儿童处于危险环境
- 婴儿哭声开场
- 未成年人参与剧情

**解决方案**:
- 能换成年人就换成年人
- 非必须用未成年人则不做

**案例**: iPhone脚本原本有儿童版本 → 换成年人形象后依然爆

### 原则二：避开恶心/擦边内容

**禁止类型**:
- 生理性不适画面
- 擦边内容

**原则**: 封面看着就不舒服的，不要做

### 原则三：强画面落文案

**定义**: 光看画面不看字幕，也能看懂脚本在讲什么

**优点**:
- 不依赖语言
- 翻译成英文更准确
- 外国人也能看懂

**案例**: 水枪脚本(常识认知，国内外都能懂)

### 原则四：不受文化差异影响

**注意**: 避免只有国内才懂的梗

**禁止类型**:
- 高考相关段子
- 国内特定职业/背景
- 只有中国人懂的笑话

**原因**: 老外get不到笑点，报点没了

## 7. 实操复刻流程

### 整体流程

```
找链接 → AI分镜 → 调整脚本 → 生成角色图 → 生成视频 → 剪辑
```

### 7.1 用AI拆分镜头

**工具**: Google AI Studio (Gemini 3.1 Pro)

**提示词模板**: (见文档提供)

**操作步骤**:
1. 复制提示词
2. 粘贴视频链接
3. 发送，让AI分析剧情
4. 检查剧情描述是否准确
5. 不准确则补充说明
6. 继续生成得到分镜表

**重要**: 务必使用 **Gemini 3.1 Pro** 模型，不要用3flash

### 7.2 检查与调整分镜

#### 必须检查的4个问题

| 问题 | 处理方法 |
|------|----------|
| AI多加的分镜 | 直接删除 |
| 缺少的分镜 | 让AI补 |
| 台词人名不对 | 手动修改 |
| 镜头类型不对 | 改成合适的 |

#### 调整技巧

**案例**: iPhone脚本调整
- 第7个分镜多余(原视频没有制作纸杯电话的过程) → 删除
- 第6个分镜用推镜头更好(展现委屈情绪) → 拉→推
- 缺少展示悲伤表情的分镜 → 让AI补

#### 标记方法
- 删除内容: 红色删除线
- 新增内容: 黄底标注

### 7.3 生成角色参考图

**工具**: 即梦官网

**流程**:
1. 用百度/谷歌搜IP名称
2. 找全身图片
3. 复制到即梦
4. 用提示词生成参考图

**提示词模板**: (见文档)

**问题解决**:

| 问题 | 解决方案 |
|------|----------|
| 版权问题 | 换衣服(用豆包生成乞丐穿搭) |
| 人脸问题 | 转彩色素描图 |

### 7.4 生成视频片段

**工具**: 即梦2.0

**方法**:
1. 10个分镜为一组截图
2. 上传截图作为参考
3. 写分镜对应的提示词
4. 指定镜头类型

**提示词格式**:
```
[镜头1描述] | [镜头2描述] | [镜头3描述]
图片1演绎图片1中的剧本
```

**重要**: 必须写镜头类型，否则会按参考图画风

### 7.5 补充分镜

**场景**: AI漏掉的镜头

**方法**:
1. 上传环境图(保持一致性)
2. 上传2张角色参考图
3. 指定9:16比例
4. 用即梦1.0老模型生成(避免卡顿)

## 8. 剪辑技巧

### 8.1 准备BGM

#### 方法一：从对标视频提取

**步骤**:
1. 打开对标视频
2. 点击视频下方音乐信息
3. 复制视频下载
4. 导入剪映分离音频
5. 只使用BGM

#### 方法二：平台识别

**适用**: 对标视频有版权音乐

**平台**:
- QQ音乐
- 网易云音乐
- 剪映音乐库

**方法**: 听歌识曲 → 搜索 → 下载

#### 方法三：问AI

**操作**: 把视频丢给AI问BGM名称

**进阶**: AI识别不出，让AI写提示词用Suno生成

### 8.2 对齐音乐节奏

**操作**:
1. 打开对标视频听开头
2. 确定从音乐第几秒开始
3. 剪映中与对标对齐
4. 删除原视频平台原因

### 8.3 剪辑注意事项

| 环节 | 建议 |
|------|------|
| BGM | 先导入，对齐节奏 |
| 转场 | 不同场景间加转场 |
| 调色 | 剪映智能调色 |
| 字幕 | 加字幕(导出1080) |
