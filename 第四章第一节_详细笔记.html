简单形式类脚本: - 直接用官方手册的脚本截图大法 - 参考: 深海圈即梦2.0信息分享汇总帖

💡 关键技巧

为什么一定要有key frame? - 即梦视频参考图片的能力很强 - 可以完美继承关键帧图片角色的服装、角色站位、场景细节、图片的风格 - 用关键帧图可以提前固定住整个视频的风格基调 - 图片抽卡成本远远低于视频抽卡

key frame怎么选? - 核心原则: 优先选中景为主、能清晰交代环境背景的图 - 中景既能展现人物主体的动作或状态,又能包含足够的背景信息 - 它就像电影里的主镜头,奠定了这一段情节的视觉基调

🎬 案例展示

案例1:周星驰《家有喜事92》经典整蛊脚本复刻 - 使用Seedance 2.0生成两段15秒的视频 - 后期精调即可

案例2:Rumi被摸PP超短脚本改造 - 更简单,生成一段15秒视频 - 后期精调即可

📝 方法分享

(1) 打开脑洞 - 不要完全依赖AI给微调方案 - AI给的方案通常会天马行空,不切实际 - AI也给很多人提供了完全一样的方案 - 在对人的情感的把控方面,AI现阶段不可能达到人类的水平 - 要大胆想象,让Seedance 2.0帮我们实现

(2) 做好前三秒 - 整体脉络最重要,决定观众是否会点赞、分享 - 前三秒极其重要,如果节奏拖沓没有看点,观众会划走 - 前三秒关键点: - 信息密度要高 - 画面要一直有变化 - 多切换镜头视角和景别 - 安排更多的动作序列和多变的人物表情

(3) 后期精调 - 删减冗余画面 - 调速(一般加快,某些情况慢动作) - 加音效(更生动) - 加特效(片段表现力不够时) - 配符合情景的背景音乐 - 加转场(不同场景间) - 画面智能调色(剪映自带)

注意事项: 导出后用视频播放器播放确认效果

(4) 多号齐发 - 同一脚本多账号分发,爆率提升几倍 - 建议做一下对白或部分细节的调整 - 流量这玩意带点玄学,好脚本在不同号表现不同

2. 语言类赛道

2.1 为什么要做AI语言类shorts

🎯 三大理由

1. 无语言shorts仍然是基础 - 生财里无语言类shorts已无门槛 - 不建议通过AI语言类shorts来开YPP(难度较大)

2. 更大的赛道容量,更强的变现能力 - 老马教练: "万物皆可解说" - 张九月: "流量是品味的敌人,品味是单价的朋友"

RPM对比: | 类型 | RPM | |------|-----| | 无语言KPOP (1100w播放) | ~0.01-0.03 | | 语言类KPOP (800w播放) | ~0.09-0.15 | | 差距 | 5-12倍 |

语言人群分布: - 英语: 美国人、菲律宾等 - 西班牙语: 墨西哥 - 葡萄牙语: 巴西 - 无语言: 印度人最多

3. 制作难度并不会高

2.2 AI语言类shorts的类型

类型一:轻度对话

特点: - 无语言shorts加上几句简单的旁白、内心戏 - 配音工作量不高

制作方法: 1. 视频制作完成后丢给GAS 2. 让GAS结合剧情写适当的台词 3. 使用ElevenLabs、Minimax等配音 4. 丢剪映里剪辑 5. 或使用Veo3、海螺V3.1等模型直接生成音画同步内容

案例: 美人鱼脚本 - 开头三句旁白 - 中间一句 - 结尾一句

类型二:中度对话

特点: - 较多的对话,但对话不长 - 语言含量中等

案例: 抽象腿视频(500万播放) - 约30个分镜 - 20个分镜有台词 - 15个分镜对的上口型 - 约110个词

风险提示: ⚠️ 名人猎奇视频有很大风险 - DEBUFF叠满: 名人猎奇、密集恐惧、儿童 - 一周五亿流量,颗粒无收

制作要点: - 很片段时长在1s-2.5s范围 - 台词简短,能更好地融入无语言shorts

类型三:重度对话

特点: - 全程对话,语速较快 - 语言含量很高

案例: Speed帮助大家但没人帮他 - 约22个分镜 - 220个词 - 一分钟之内说完

语速要求: - 每分钟130-180词 - 比普通人平均语速(60-100词/分钟)快 - 属于正常演讲、快读范畴

制作要点: - 只能在说话同时做简单动作 - 做一些表情变化丰富信息密度

2.3 实操步骤

第一步:提取文本,生成音频

1. 提取文案: - 把链接和要求发给GAS - 或使用哼哼猫等下载软件下载字幕

2. 更换角色,修改内容: - 不做儿童,把爸爸妈妈儿子女儿改成同龄人 - 例如: Sakura→Rumi, Naruto→Abby, Mom→Mira, Dad→Jinu - 检查修改后的内容是否和情节符合

3. 导出JSON: - 让GAS按要求导出JSON - 去will大佬的网站生成音频: aihotvideos

ElevenLabs配音提示词:

# Role: YouTube字幕处理专家 & ElevenLabs架构师 (v17.0 Master)

核心原则:全流程采用双语对照表格形式展示

Phase 1: 角色对话提取与翻译
- 精准角色识别
- 逐字原文提取
- 高质量翻译

Phase 2: 配音脚本标准化
- 格式清洗:添加[Emotion]开头,去除句尾标点
- 互动植入

Phase 3: 最终生产(JSON Production)

第二步:生成图片

方法: 1. 按照小黄教练的故事板提示词和分镜版来复刻 2. 把提示词复制到飞书表格,导出CSV格式 3. 使用工具: 故事板分镜台词处理.zip 4. 转换成JSON后去aihotvideos生成图片

第三步:生成视频

提示词模版: CineDream Architect v12.0 (The Omnibus)

核心要素: - 角色库: Rumi、Mira、Zoey、Jinu、Abby、Mystery - 绝对禁令: 严禁画外描写、严禁运镜速度词、严禁弱动词 - 运镜手册: 推进、拉远、跟随、环绕、摇移、固定 - 表情与情绪库: 复合情绪词 - 音效与环境库: 自然音效、人造音效、动作音效、环境底噪 - 光影与风格库: Golden hour、Blue hour、Cyberpunk neon等

核心铁律: 1. 光栅锁死法则: 所见即所得 2. 6秒动作链: [起始动作] -> [发展动作] -> [结束/反应] 3. 全感官音频协议: 音效与画面精确对应 4. 核心提示词公式: [运镜方式], [视觉主体] + [强动作链], [表情弧光], [环境/光影], [Dialogue], [SFX], [Ambient]

第四步:半自动剪辑

工具: - AutoFlow.zip (谷歌插件) - 文件夹生成器.zip - capcut auto-v1.0.0.zip

配置: - 修改config.json里的素材根目录和剪映草稿路径 - 可修改音频倍速等内容

效果: - 最主要的是音频剪了气口,自动对齐 - 视频还是要自行剪辑

2.4 小结

关键要点: ✅ 相对于无语言shorts,AI语言类制作难度并不大
✅ 没有那么多分镜,不需要用特别快的节奏
✅ 只要故事足够吸引人,观众愿意看下去
✅ 同质化低,不在自己的赛道里卷
✅ 前三秒依然重要,频道平均选择观看率至少要80%
✅ 洗稿,不要照抄
✅ 音效要加上去
✅ 配音保持频道特色
✅ 加字幕,导出1080

6. 科普解说(拼接类解说)

6.1 什么是科普解说

概念理解: - 不是科学科普、地理人文讲解 - 而是"十万个为什么" - 没见过的东西、现象、事物都可以解说

拼接类解说: - 通过选题采集围绕选题的不同素材 - 通过拼接放在一起的手法 - 属于科普解说的一种

优缺点: | 优点 | 缺点 | |------|------| | 基本不会出现0播、低播 | 太稳重,天天万播 | | 标准化流程后各个视频都是万播 | 流量比爆款视频解说低 | | 单兵作战比较强劲 | 胜在稳定 |

6.2 制作流程

第一步:产出文案

正练方式:

1. 确定选题: - 看科普解说类型的对标内容 - 选择10-50个文案丢给克劳德 - 让AI理解并输出这类型的选题是什么

2. 创造选题:

提示词模版:

请帮我创作[数量]个shorts video选题创意,要求如下:

内容结构:
每个选题必须包含:
- 主题标题
- 完整分行文案(150到200英文,200-300字中文)
- 中文关键词5个(用于素材搜索)
- 英文关键词5个(用于素材搜索)
- 选题类型说明(科普/文化/生活等)

文案要求:
- 每句话需要分行书写
- 语言通俗易懂
- 突出新奇性和知识性
- 结尾加入互动引导
- 适合30-50秒短视频呈现
- 文案要先给到完整英文,再给到完整中文

选题标准:
- 话题具有新鲜感和传播性
- 容易获取相关视频素材
- 适合视觉化展示
- 能激发用户评论互动
- 符合shorts video平台调性

关键词标准:
- 关键词要具体且实用
- 符合各平台搜索习惯
- 中英文关键词相对应
- 便于实际素材搜索
- 尽量选择高搜索量词汇

目标受众:[目标年龄段]
传播平台:[具体平台]
内容领域:[具体领域]
文案风格:[提供情绪]

3. 判断选题是否可做: - 根据关键词搜索是否有相关素材 - 配合文案感受是否合适

重要提示: ⚠️ 拼接类剪辑素材可以直接在YouTube上寻找并混剪 ⚠️ 只要不把原作者的脸露出来,大概率不会有问题 ⚠️ 这是合理利用版权,美国版权法第107条允许

4. 文案二次训练:

目标: 1. 开头前三秒留住人,具备夸张性开头特点 2. 后面的部分口语化,简洁明朗没有废话

方法: 把写出来的文案丢进去套用模版二次进化

逆练方式:

第一步:保存对标视频的文案

第二步:转录成英文 - 使用飞书妙记 - 或其他AI转录工具

第三步:改写

第二步:根据文案找对应素材

拼接类素材选择的注意事项: 1. 长视频也可用,某一个亮点片段下载下来单独裁剪 2. 在YouTube中搜索时可以不用点进去,鼠标放在视频上预览

素材寻找方法:

方法1:平台主动搜索关键词 - 提炼关键词 - 根据文案逐段搜索 - 如果AI提炼的关键词有问题,尝试复制原文第一段话搜索

方法2:谷歌识图功能/百度识图 - 把想要画面的截图丢给谷歌识图 - 找到很多素材 - 尽量不用同行的视频素材

拓展搜索: - 热钢卷、冷却钢卷、熔炼钢卷等 - 让GPT给词,或中文翻译成英文直接搜索

6.3 剪辑优化

第一步:优化转场、气口

问题: - 视频直接素材拼接,加配音 - 素材找的不符合,画面看不清 - 没有转场特效

解决方案: 1. 识别字幕 2. 按每句话的停顿添加画面 3. 每句话结束就转场下一个画面 4. 一句话较长,中间多加几个画面 5. 添加转场特效、音效

效果: 视频质感上升,流量达到万播标准值

总结: ⚠️ 一定要认真学习手册,一步一步对照着来 ⚠️ 不要有自己的想法 ⚠️ 如果达不到标准值,回头看看动作是否变形

第二步:优化开头前3s

YouTube指标: - 平均观看百分比 - 选择观看率

操作步骤: 1. 把对标视频的开头关键帧截图 2. 谷歌识图功能搜图,找到原视频 3. 下载视频,找到和对标视频同样的片段 4. 看对标视频放了几秒,咱们就跟上 5. 对标用了一个箭头,咱们也用上一个箭头 6. 尽量做到非常相似

重要提示: ⚠️ 把开头和对标视频做的一模一样 ⚠️ 文案的开头也不改动 ⚠️ 这样来提升选择观看率(非常重要)

成果: - 优化后迎来第一个爆款 - 720w观看次数 - 发布一星期才开始爆

总结: ⚠️ 不要看到数据不好就焦虑、断更 ⚠️ 多从自己身上找原因 ⚠️ 坚持更新,等待系统推流

6.4 配乐

音乐选择的重要性: - YouTube对版权保护有严格规定 - 随意添加有版权音乐可能面临警告 - 超过3次警告,频道可能被封禁 - 视频收益全部归音乐版权方所有

无版权音乐来源:

1. YouTube音乐库 - Shorts: 上传视频时点击"添加音效" - 长视频: YouTube工作室 → 音频库 - 最安全的选择

2. 爱给网 - 网址: https://www.aigei.com/ - 配乐 → 免费商用(cc协议) - 需要看清楚引用要求 - 需要署名的在视频说明中写明来源

署名格式:

本视频中使用的音乐名称:XXXX
作者:XXXX
来源:https://www.aigei.com/music/song/

3. SoundCloud - 网址: https://soundcloud.com/ - 搜索栏搜索关键词 - 选择"Tracks" → "To listen to" → "To use commercially" - 没有下载入口可使用SoundCloud Downloader工具

4. 优质的YouTube音乐频道 - ROYALTY FREE MUSIC - Audio Library - FreeMusicWave - No Copyright Music - Vlog No Copyright Music

版权规避注意事项: - 最安全: YouTube音乐库 - 其他渠道: 当前无版权,但创作者可能未来注册版权 - YouTube会扫描并转移收益给版权方 - 规避风险: 找付费版权音乐网站 - 自己创作或AI工具(付费版Suno)制作的音乐归自己

7. 爆款视频解说(原视频解说)

7.1 什么是爆款视频解说

定义: 以爆款视频深度二创去重,或一个博主的多个视频剪辑在一起

优缺点: | 优点 | 缺点 | |------|------| | 借鉴的都是爆款视频 | 可能遇到博主diss | | 二创去重后流量更好 | 版权警告、要赔偿 | | | 要求删掉视频 |

规避方法: - 要么不以人物为主体 - 要么以国内爆款素材为主体 - 避免很多风险

7.2 制作流程

第一步:找到爆款视频

不需要AI出选题,需要寻找已有爆款视频

素材来源: - 国内: 抖音、快手 - 海外: 推特、INS、TikTok

建立素材库: - 做爆款视频解说: 抢爆款视频的二创首发机会 - 做拼接类解说: 收集同类型题材的素材

平台采集: - 锚定TikTok、X、INS - 复选国内抖音、快手 - 国内素材加入文字多,去重去水印麻烦

了解赛道属性: ⚠️ 海外人物视频素材做解说面临侵权风险 ⚠️ 遇到小气的素材作者可能找麻烦 ⚠️ 爆款视频解说优先选择国内爆款视频素材 ⚠️ 科普解说拼接类素材首选海外平台

举例: X平台: 1. 确定搜索内容如"Animals" 2. 在Top里发现爆款视频 3. 检查是否是同行 4. 如果是内容创作者,关注点赞 5. 挖掘更多作者 6. 第二天打开主页推荐,获得更多内容素材

不同平台特点: - TikTok和INS: 内容和YouTube shorts高度相似,少选择 - 可用作补充素材或监控作者抢二创首发权

沉淀作者方法: - 搜索关键词,关注同类型账号 - 休息一会儿或第二天通过推荐页面刷新 - 最好是新号,容易建立信息茧房 - 不要用网页刷,用手机刷

第二步:准备文案

① 正练文案: - 手动写描述词 - 或把视频抽出图片发给GPT/克劳德 - 让AI描述整体情节 - 套用解说词模版完成文案二次进化

第三步:剪辑

准备素材: 1. 改写后中文文案 2. 对标视频的原视频素材 3. 改写后配音

① 导入音频,并识别字幕 - 导入音频时切气口

② 导入原视频做处理

解说词与视频时长匹配: - 解说词比视频长: 调慢原视频速度,或反复复制片段镜像倒放补充时长,或通过类似片段代替 - 解说词比视频短: 删减片段,或加速匹配音频

③ 处理原视频 1. 智能场景分割 2. 选择镜像 3. 挨个对每个片段进行居中显示 4. 通过裁剪去除字体符号 5. 画中画效果(背景填充-模糊) 6. 全部处理完后创建复合片段 7. 调整对齐音频时间轴 8. 通过加减速方式

④ 给复合片段智能调色 - 智能调色(曲线调整,锐化等) - 锐化参数 - 色温饱和度色调 - 曲线-亮度-红色通道-蓝色通道-绿色通道 - 原素材音效能提升视频感觉可放出来一些

⑤ 调整字幕特效

关键: 原素材视频解说的重点

原因: - 原素材要进行视频深度混剪 - 如果有大量使用这个素材的创作者,可能会遇到0播问题 - 没有过重 - 字幕特效是很好的过重方式
